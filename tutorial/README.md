Want an easy way to play around with YAKL on your laptop without needing a GPU or having to know CMake? This tutorial directory is for you! You only need a C++ compiler to run these examples on your CPU and play around with YAKL and get the feel for it. YAKL is a header-only library in C++, and you only have to have include directory flags when compiling. There are many other flags you can use to change YAKL's behavior, but you do not need them. 

Please check out the [diffusion](./diffusion) directory first and then try out the other directories as well:
* [Diffusion](./diffusion/README.md)
* [Shallow and Deep Copy](./shallow_deep_copy/README.md)
